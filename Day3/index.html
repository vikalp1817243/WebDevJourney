CSS Selectors

CSS selectors are used to select the HTML elements that you want to style. There are several types of CSS selectors, including:

  • Element Selector: Selects all elements of a specific type. It applies to all instances of that element. Example: p { color: red; }
  • Class Selector: Selects all elements with a specific class. It can be used multiple times on a page. Example: .myClass { color: blue; }
  • ID Selector: Selects a single element with a specific ID. It should be unique within the page. Example: #myId { color: green; }
  • Attribute Selector: Selects elements based on their attributes. Example: a[target="_blank"] { color: orange; }
  • Universal Selector: Selects all elements on the page. Example: * { margin: 0; padding: 0; }
  • Pseudo-class Selector: Selects elements based on their state. Example: a:hover { color: purple; }
