CSS Display

CSS display is a property that determines how an element is displayed on a webpage. It can be used to control the layout and visibility of elements.

Types of CSS Display

There are several types of CSS display that you can use to control the layout of your webpage:

  • Block
  • Inline
  • Inline-block
  • None

1. Block Display

Block display is the default display for most elements. It means that the element will take up the full width of its container and will start on a new line.

Let's look at an example using code. Lets start with html code:

<div class="block"> <p>This is a block element</p> </div> 
            .block {
                display: block;
            }

2. Inline Display

Inline display means that the element will only take up as much width as necessary and will not start on a new line.

Look at the following example:

1. Hello, this is an inline element. 2. Hello, this is an inline element. 
            .inline {
                display: inline;
            }

Note:You cannot set the height and width of these elements beacuse they are default to thesize of content. Also since you cannot apply this property to div elements because they are block-level elements. When you set a div to display: inline, it stops behaving like a box. Inline elements ignore width and height, so if the element is empty, it has almost no painted area, which makes the background color look like it is not applied. The background only becomes visible around actual content or padding. If you want it to show as a colored box, use display: inline-block or display: block. If you want it to stay inline, add content or padding.

3. Inline-Block Display

This display type combines the properties of both block and inline elements. It allows you to set the height and width of the element while still allowing it to sit inline with other elements.

Look at the following example:

1. This is an inline-block element. 2. This is an inline-block element.

.inline-block { display: inline-block; }

4. None Display

None display means that the element will not be displayed on the webpage. It will not take up any space and will not be visible to the user.

Look at the following example:

.none { display: none; }
