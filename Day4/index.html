Parent selector

Parent selector is a CSS selector that allows you to select an element based on its parent element.

This will select all <li> elements that are direct children of a <ul> element and apply the color red to them. The parent selector is useful for targeting specific elements in a nested structure without affecting other elements that may be similar but are not direct children.
