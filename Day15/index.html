Flex-sizing

Flex items can be sized using the flex property, which is a shorthand for flex-grow, flex-shrink, and flex-basis. The flex property allows you to specify how much a flex item should grow or shrink relative to the other items in the container. and it should behave in responsive situation.

Let look at cenario

Lets we have 4 paragrah emelments on page without any css. They will normally display in a row according to their content size.

if those paragraph elements are placed inside a flex container, their sizing behavior will be different and by default they lie on x-axis wihtout any space

Browser behavior look at the follwoing priorty oder givne below

Lets look at the priority list:

Content < Width < flex-basis < min-width/max-width

In this order, the browser will apply each constraint to determine the final size of the flex item.
