Attributes

Attributes are used to provide additional information about HTML elements. They are specified within the opening tag of an element and can be used to modify the behavior or appearance of the element.

Common Attributes

  • id: Used to uniquely identify an element.

    • Example: id="myElement"

  • class: Used to classify elements into groups for styling purposes.

    • Example: class="myClass"

  • src: Specifies the source of an image or media file.

    • Example: src="image.jpg"

  • href: Specifies the URL of a link.

    • Example: href="https://www.example.com"

  • alt: Provides alternative text for images.

    • Example: alt="Description of the image"

  • title: Provides additional information about an element, often displayed as a tooltip.

    • Example: title="Additional information"

  • style: Allows you to apply inline CSS styles to an element.

    • Example: style="color: red; font-size: 16px;"

  • data-*: Custom attributes that can be used to store extra information about an element, which can be accessed via JavaScript.

    • Example: data-user-id="123"

How to add a favicon icon?

To add a favicon icon to your HTML document, you can use the <link> tag in the <head> section. The <link> tag should have the rel attribute set to "icon" and the href attribute set to the path of your favicon file.

<link rel="icon" type="image/x-icon" href="./favicon.ico">

Make sure to replace "./favicon.ico" with the actual path to your favicon file. The favicon will then be displayed in the browser tab when users visit your website.

The rel attribute specifies the relationship between the current document and the linked resource.

Type: image/x-icon specifies the MIME type of the favicon file.
