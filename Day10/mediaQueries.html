Media Queries

Media queries allow you to apply styles to different devices based on their screen size, orientation, resolution, etc.

Example of media queries:

@media (max-width: 600px) { body { background-color: lightblue; } }

When the screen width is less than 600px, the background-color of the body changes to lightblue.

Commonly used properties of media queries:

  • min-width
  • max-width
  • min-height
  • max-height
  • orientation
  • resolution

Breakpoints

Breakpoints are the points at which the styles of the website change based on the screen size.

Commonly used breakpoints are:

  • 600px
  • 768px
  • 992px
  • 1200px

Mobile-first approach

Mobile-first approach is a responsive design approach that involves designing for mobile devices first and then scaling up for larger devices.

Commonly used properties of mobile-first approach are:

  • min-width
  • max-width
  • min-height
  • max-height
  • orientation
  • resolution

Desktop-first approach

Desktop-first approach is a responsive design approach that involves designing for desktop devices first and then scaling down for smaller devices.

Commonly used properties of desktop-first approach are:

  • min-width
  • max-width
  • min-height
  • max-height
  • orientation
  • resolution

In conclusion, media queries are a powerful tool that can be used to create responsive websites. They are a simple and effective way to apply different styles to different devices based on their screen size, orientation, resolution, etc.

Use them wisely and you will be able to create responsive websites that look great on all devices.
