Group Selectors

Group selectors allow you to apply the unique styles to multiple elements at once. In order to do so multiple class names can be specified in the tags. separated by commas.

                .class1, .class2 {
                    color: red;
                }

In the above example, both elements with the class "class1" and "class2" will have their text color set to red.

Child selectors

Child selectors allow you to target elements that are direct children of a specific parent element. This is done by using the ">" symbol between the parent and child selectors.

Look at the following example of html code:

                <div class="parent-class">
                    <p>This is a direct child 1</p>
                    <div>
                        <p>This is not a direct child</p>
                    </div>
                    <p>This is a direct child 2</p>
                </div>

In the above example, there are two paragraph elements that are direct children of the div with the class "parent-class", and one paragraph element that is nested within another div. If we apply the following CSS selector: 

                .parent-class > p{
                    color: blue;
                }

In the above example, only the paragraph elements that are direct children( only one level down) of an element with the class "parent-class" will have their text color set to blue.

Descendent selectors

Descendent selectors allow you to target elements that are nested within other elements. This is done by using a space between the parent and child selectors. It can go multiple levels deep.

Look at the following example of html code:

                <div class="outer-class">
                    <div class="inner-class">
                        <p>This is a descendent</p>
                    </div>
                </div>

In the above example, the paragraph element with the class "inner-class" will have its text color set to red.

                .inner-class p{
                    color: red;
                }

In the above example, the paragraph element with the class "inner-class" will have its text color set to red.

Chain Selectors

Chain selectors allow you to target elements that have multiple classes applied to them. This is done by concatenating the class names without any spaces. Note that the order of the classes in the selector does not matter. Also they should follow theri own intialisation rules. Like .class#id.class{} is an example of a chain selector. Note the use of the hash symbol to target an element with a specific ID.

Look at the html code:

                <div class="class1 class2">
                    This element has both class1 and class2
                </div>
            
                .class1.class2 {
                    color: green;
                }

In the above example, only elements that have both the "class1" and "class2" classes will have their text color set to green.
