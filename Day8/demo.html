1️⃣ Static Positioning (Default)

Static positioning is the default for all elements. They follow the normal document flow.

Key Points:
  • Default position value
  • Elements follow normal document flow
  • Top, right, bottom, left properties are ignored

2️⃣ Relative Positioning

Relative positioning moves an element relative to its normal position, but the space it originally occupied is still reserved.

Key Points:
  • Element positioned relative to its normal position
  • Original space is preserved in document flow
  • Use top, right, bottom, left properties for offset
  • Other elements are NOT affected

3️⃣ Absolute Positioning

Absolute positioning removes the element from normal flow and positions it relative to its nearest positioned ancestor.

Key Points:
  • Removed from normal document flow
  • Positioned relative to nearest positioned parent (or viewport)
  • Space occupied by element is NOT preserved
  • Perfect for overlapping layouts

Bottom-Right

.absolute-container { position: relative; /* Creates positioning context */ } .absolute { position: absolute; top: 10px; left: 10px; }

4️⃣ Fixed Positioning

Fixed positioning removes the element from normal flow and fixes it to the viewport. It stays in place even when scrolling.

Key Points:
  • Positioned relative to viewport
  • Stays in place during scrolling
  • Removed from document flow
  • Common for navigation bars, floating buttons
✅ A fixed button appears at the bottom-right corner. Scroll the page to see it stay in place! (Orange button)
.fixed { position: fixed; bottom: 20px; right: 20px; }

5️⃣ Sticky Positioning

Sticky positioning is a hybrid of relative and fixed. It acts as relative until it reaches a threshold, then acts as fixed.

Key Points:
  • Starts as relative positioning
  • Becomes fixed when scrolling reaches threshold (top/bottom value)
  • Stays within parent container (doesn't scroll beyond it)
  • Useful for sticky headers, sidebar menus

6️⃣ Z-Index (Stacking Order)

Z-index controls the stacking order of positioned elements. Higher values appear on top.

Key Points:
  • Only works with positioned elements (not static)
  • Higher z-index values appear on top
  • Default value is 0
  • Can use negative values
  • Order in HTML matters when z-index is equal
z: 1
z: 3 (Top)
z: 2
.box-1 { position: absolute; z-index: 1; } .box-2 { position: absolute; z-index: 3; } /* Appears on top */ .box-3 { position: absolute; z-index: 2; }
