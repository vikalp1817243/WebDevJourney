Z-Index

The z-index property controls the stacking order of elements. Elements with a higher z-index value will be displayed on top of elements with a lower z-index value.

How Z-Index Works

Z-index only works on elements that have a position value of relative, absolute, fixed, or sticky. The default z-index value is 0. If two elements have the same z-index value, the one that appears later in the HTML code will be displayed on top.

Look at the example below:

            .element1 {
                position: relative;
                z-index: 1;
            }
            .element2 {
                position: relative;
                z-index: 2;
            }

In the above example, element2 will be displayed on top of element1 because it has a higher z-index value.
