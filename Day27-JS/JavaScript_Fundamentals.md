# JavaScript Fundamentals — Concise Notes

Quick, practical notes mixing theory + code + examples for fast recap.

---

## Quick Cheatsheet
- Variables: `let`, `const`, `var` (avoid `var`).
- Types: `Number`, `String`, `Boolean`, `Null`, `Undefined`, `BigInt`, `Symbol`, `Object`.
- Equality: `==` (coercing), `===` (strict).
- Functions: declarations, expressions, arrow functions.
- Arrays/Objects: `[]`, `{}`; use array methods like `map`, `filter`, `reduce`.
- Promises: `.then/.catch` or `async/await`.
- Modules: `export` / `import`.
