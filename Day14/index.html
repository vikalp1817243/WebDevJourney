Flexbox-Layout

About

Flexbox is a layout model that allows elements to be aligned and distributed in a predictable way.

It provides a more efficient way to lay out, align and distribute space among items in a container, even when their size is unknown or dynamic.

flexbox allows for easy alignment and distribution of items within a container.

Features

  • Flexible item sizes
  • Easy alignment and distribution
  • Responsive design capabilities
  • Supports both horizontal and vertical layouts

Important Note

The properties listed here only applies to child elements of the flex container.

The flex container is the parent element that has the display: flex; property applied to it.

List of flex properties

  1. order: controls the order of flex items
  2. flex-direction: controls the direction of the flex container
  3. flex-wrap: controls whether flex items wrap or not
  4. flex-flow: a shorthand for flex-direction and flex-wrap
  5. flex-grow: controls how much a flex item can grow
  6. flex-shrink: controls how much a flex item can shrink
  7. flex-basis: controls the initial size of a flex item
  8. justify-content: controls the alignment of flex items along the main axis
  9. align-items: controls the alignment of flex items along the cross axis
  10. align-content: controls the alignment of flex lines along the cross axis
