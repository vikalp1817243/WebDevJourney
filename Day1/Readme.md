# DAY 1
## LEARNINGS

- **HTML Structure**: `<!DOCTYPE html>`, `html`, `head`, `body` — the basic skeleton of a webpage. Put metadata and the `title` in `head`, and visible content in `body`.

- **Links (`a href`)**: `href` sets the destination URL. Use `target="_blank"` to open the link in a new tab. Example: `<a href="https://example.com" target="_blank">Visit</a>`

- **Images (`img`)**: `src` points to the image file and `alt` provides alternative text for accessibility. Example: `<img src="mozilla.jpeg" alt="Mozilla Logo">`

- **Lists (`ul`, `li`)**: Group related items as bullet lists.

- **Inline Styles (`style` attribute)**: Add quick CSS directly to elements, e.g. `<h1 style="text-align:center">Title</h1>`. For larger projects prefer external CSS files.

- **Flexbox (layout)**: `display: flex` makes a container flexible. Common properties:
	- `justify-content`: controls horizontal distribution (e.g., `space-between`).
	- `gap`: space between flex items.
	- `flex` on children (e.g., `flex: 1`) makes items grow/shrink.
	Example: `<div style="display:flex; gap:20px;"><div style="flex:1">A</div><div style="flex:1">B</div></div>`

- **Text alignment**: `text-align: center|left|right` aligns inline content inside a block.

- **Footer**: Use a `footer` element for page footer content; style it as needed (background color, text color, padding).

- **Background vs Background-Color**: `background` is a shorthand that can set color, image, position, and repeat (e.g., `background: red url(img.png) no-repeat center;`). `background-color` only sets the solid color (e.g., `background-color: #f0f0f0;`).

## QUICK USAGE NOTES

- Prefer external CSS for repeated styles.
- Use meaningful `alt` text for images for accessibility.
- Use `flex` for simple responsive horizontal layouts instead of floats.
